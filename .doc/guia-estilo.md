# Guia de Estilo

- Surface: `#102C48`
- Border: `#1E3A56`

## Tipografia
- Títulos (serif): **Cormorant Garamond** (fallback Georgia)
- Texto (sans): **Inter** (fallback Segoe UI/Roboto)
- Mono: stack do sistema (Menlo/Consolas…)

## Tokens (SCSS)
Variables em `src/styles/variables.scss` + utilitários em `mixins.scss`.
